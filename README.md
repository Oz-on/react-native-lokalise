# react-native-lokalise

React Native wrapper for Lokalise SDK

## Installation

```sh
npm install react-native-lokalise
```

## Usage

```js
import Lokalise from "react-native-lokalise";

// ...

const result = await Lokalise.multiply(3, 7);
```
